Curso de Finanzas para emprendedores

Incorporá las principales herramientas financieras a la gestión diaria.

El curso de Finanzas para Emprendedores brinda las herramientas necesarias para lograr una gestión financiera eficiente.

    En este curso aprenderás:
  • Cómo separar las finanzas personales de las finanzas de tu negocio.
  • Cómo armar un presupuesto financiero.
  • Cómo desarrollar un flujo de caja y proyectar balances financieros.
  • Cómo medir el crecimiento de tu negocio.
  • Desiciones de inversión.
